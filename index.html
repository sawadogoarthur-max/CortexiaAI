Session 1 - Médias

Quelle est la durée de la formation 'Maîtriser la Data et l'IA dans les Médias' ?
La formation dure 3 jours et se déroule en ligne de 18h à 21h.
Quels sont les objectifs de la formation pour les professionnels des médias ?
Cette formation permet de maîtriser le Data Storytelling, créer des présentations data-driven, utiliser l'IA pour générer du contenu journalistique et comprendre l'éthique et les limites de l'IA dans les médias.
Que vais-je apprendre dans le module 1 sur les fondamentaux du Data Storytelling ?
Vous apprendrez : qu'est-ce que le Data Storytelling, le schéma narratif et sa construction, les caractéristiques d'une bonne histoire, la méthode AIDA (Attention, Intérêt, Désir, Action), développer votre esprit créatif, analyser le contexte et l'audience, construire votre histoire avec des données.
Comment vais-je apprendre à illustrer mon histoire avec des visuels ?
Dans le module 2, vous étudierez : la théorie de Gestalt et perception visuelle, comment choisir un contenu visuel pertinent, éviter le désordre visuel, modèles visuels, infographies, templates et datavisualisation, utiliser la vidéo narrative, exercices pratiques et études de cas.
Que vais-je apprendre sur les présentations d'entreprise ?
Dans le module 3, vous apprendrez : construire une présentation de données impactante, adapter le message à l'audience interne ou externe, renforcer la crédibilité avec des données, études de cas sur la communication des résultats, atelier pratique : créer une mini-présentation data-driven.
Qu'est-ce que l'art du Prompting et pourquoi est-ce important ?
Le module 4 vous apprend à dialoguer efficacement avec l'IA pour produire du contenu pertinent et original. Vous apprendrez : le rôle du prompt et son interprétation par l'IA, différentes formes de prompting (descriptif, conversationnel, créatif, structuré, etc.), techniques avancées : modèles réutilisables, personas IA, gestion des biais, applications concrètes : génération de titres, scripts, articles, visuels, résumés, atelier pratique : comparaison et analyse de prompts, construction d'une bibliothèque de prompts.
Que vais-je apprendre sur l'IA et le journalisme de données ?
Dans le module 5, vous apprendrez : histoire et évolution de l'IA dans les médias, prise en main de ChatGPT, Mistral, HuggingFace et agents IA, principes de prompt engineering et applications pratiques, vérification et analyse de contenu assistée par IA (fact-checking, détection de contenu généré par IA), éthique, protection des données, droits d'auteur et collaboration IA-journaliste, tendances futures : LLMs, IA multimodale, IA de rédaction, atelier final : mini-enquête journalistique assistée par IA.

Session 3 - Vibe Coding

Quelle est la durée de la formation 'Vibe Coding : création et automatisation avec les agents IA' ?
La formation dure 3 jours et se déroule en ligne de 18h à 21h.
Que vais-je apprendre dans le module 1 sur le Vibe Coding et les agents IA ?
Vous apprendrez : bases du Vibe Coding et des agents IA, évolution de l'IA : analytique → générative → autonome, différences entre agents IA et IA générative, définition et anatomie d'un agent : perception, mémoire, raisonnement, action, exemples concrets : agents de réservation, copilotes RH, gestionnaires de projets, interaction avec les systèmes : API, CRM, Zapier, etc.
Que vais-je apprendre dans le module 2 sur les principes fondamentaux de l'IA générative ?
Vous apprendrez : fonctionnement des LLM (GPT, Claude, Gemini), limites des modèles : hallucinations, bugs, manque de contexte, prompting simple, multi-étapes, incrémental et utilisation de snippets de code, bonnes pratiques de structuration des prompts.
Que vais-je apprendre dans le module 3 sur la conception et le cadrage d'un projet IA ?
Vous apprendrez : formaliser un cas d'usage IA rentable et aligné avec la stratégie de l'entreprise, identifier les risques et KPIs pertinents, méthode Problem → Impact → IA Fit et canvas de cadrage, études de cas sectorielles : RH, marketing, finance, supply chain, identification d'un projet test ou cas prioritaire.
Que vais-je apprendre dans le module 4 sur la construction d'un agent IA autonome ?
Vous apprendrez : déployer un agent capable de raisonner, interagir et déclencher des workflows, architecture Agent → Action → Mémoire, outils : LangChain, CrewAI, Autogen, critères de confiance : garde-fous, self-reflection, red-teaming, atelier fil rouge : agent traitant une demande client → GPT personnalisé → workflow n8n.
Que vais-je apprendre dans le module 5 sur le prompt au prototype fonctionnel ?
Vous apprendrez : générer du code à partir d'instructions en langage naturel, créer une roadmap sous forme de prompts, génération frontend (HTML/CSS/JS, React) et backend (Express, Flask), lire, comprendre et corriger le code généré, conserver l'historique et documentation automatique via l'IA.
Que vais-je apprendre dans le module 6 sur les limites, risques et bonnes pratiques ?
Vous apprendrez : superviser le code généré par IA, identifier les failles de sécurité, erreurs silencieuses et bugs logiques, maintenabilité du code généré, bonnes pratiques : tests automatiques, commentaires, linters, rôle du développeur : superviseur, non spectateur.
Que vais-je apprendre dans le module 7 sur le workflow Vibe Coding Pro ?
Vous apprendrez : mettre en place une collaboration homme-machine efficace, utilisation hybride avec IDE traditionnels (VS Code, Cursor, Replit), pair coding avec l'IA, gestion des changements IA + humains et anticipation des évolutions.
Que vais-je apprendre dans le module 8 sur l'atelier final 'Crée ton projet Vibe' ?
Vous apprendrez : appliquer toutes les compétences acquises sur un projet concret, concevoir un mini-projet : site web, jeu, API, extension navigateur…, définir objectifs, contraintes et prompts, génération, tests, amélioration et documentation, présentation et feedback collectif, livrables : prototype fonctionnel complet et documentation IA-assistée.

Session 4 - AI & ML

Quelle est la durée de la formation 'Artificial Intelligence & Machine Learning pour développeurs' ?
La formation dure 2 jours et se déroule en ligne de 18h à 21h.
Que vais-je apprendre dans le module 1 sur l'introduction à l'IA et au Machine Learning ?
Vous apprendrez : panorama de l'intelligence artificielle et ses applications industrielles et grand public, différences entre IA faible, IA forte, Machine Learning et Deep Learning, focus sur les réseaux neuronaux artificiels : principe, architecture et importance, impact sociétal, éthique et biais dans les algorithmes.
Que vais-je apprendre dans le module 2 sur la programmation Python pour IA ?
Vous apprendrez : environnement de développement : Jupyter Notebook, IDE, structures de données clés : listes, dictionnaires, DataFrames, manipulation de données avec NumPy et pandas : import, nettoyage, agrégation, visualisation de données de base avec matplotlib et seaborn pour interpréter les tendances.
Que vais-je apprendre dans le module 3 sur TensorFlow et PyTorch ?
Vous apprendrez : concepts de tenseurs, graphes de calcul et modèles séquentiels, création, compilation et entraînement d'un modèle neuronal simple, gestion avancée des modèles : callbacks, entraînement sur GPU, sauvegarde, comparaison et cas d'usage pratique de TensorFlow vs PyTorch.
Que vais-je apprendre dans le module 4 sur l'apprentissage supervisé, non supervisé et par renforcement ?
Vous apprendrez : algorithmes supervisés : régression linéaire, arbres de décision, forêts aléatoires, clustering k-means et ACP pour apprentissage non supervisé, introduction au deep reinforcement learning : principe et exemples, atelier pratique de classification et regroupement sur des datasets publics (MNIST, Iris).
Que vais-je apprendre dans le module 5 sur le traitement du langage naturel et la vision par ordinateur ?
Vous apprendrez : NLP : tokenization, embeddings, modèles Transformer (BERT, GPT), analyse de textes : extraction d'entités nommées, sentiment analysis, vision par ordinateur : détection et classification d'images, introduction aux CNN, projet mini : créer un classifieur d'images ou un analyseur de sentiment.
Que vais-je apprendre dans le module 6 sur le projet pratique complet ?
Vous apprendrez : collecte et préparation d'un jeu de données, implémentation complète depuis le pipeline data jusqu'à la prédiction, évaluation des performances : précision, rappel, F1-score, présentation et discussion des résultats sur des cas réels d'usage business.

Session 5 - Cloud Computing

Quelle est la durée de la formation 'Cloud Computing (AWS, Azure, GCP + DevOps)' ?
La formation dure 2 jours et se déroule en ligne de 18h à 21h.
Que vais-je apprendre dans le module 1 sur les concepts Cloud et DevOps ?
Vous apprendrez : définitions et modèles de cloud computing, avantages du Cloud comparé à l'IT traditionnelle, culture DevOps : collaboration, agilité, automatisation, études de cas réels d'implémentation DevOps.
Que vais-je apprendre dans le module 2 sur AWS, Azure et GCP ?
Vous apprendrez : présentation des offres majeures de compute : EC2, Azure VM, Compute Engine, stockage et bases de données cloud : S3, Blob Storage, BigQuery, concepts réseaux et sécurité : VPC, firewall, IAM, atelier pratique : création d'une machine virtuelle sur les consoles cloud.
Que vais-je apprendre dans le module 3 sur Terraform et Ansible ?
Vous apprendrez : principes d'Infrastructure as Code (IaC), écriture de scripts Terraform pour déployer des infrastructures répétables, automatisation de la configuration avec Ansible, exemple complet de déploiement automatisé (serveur + application).
Que vais-je apprendre dans le module 4 sur les pipelines CI/CD ?
Vous apprendrez : architecture d'un pipeline d'intégration/déploiement continu, configuration de Jenkins, GitHub Actions et GitLab CI, automatisation des tests unitaires et tests d'intégration, simulation d'un cycle complet de livraison logicielle.
Que vais-je apprendre dans le module 5 sur Docker et Kubernetes ?
Vous apprendrez : concepts de conteneurs et images Docker, création d'images Docker personnalisées et gestion des volumes et réseaux, introduction à Kubernetes : architecture et objets de base (pods, services, ingress), déploiement d'une application conteneurisée sur cluster Kubernetes local.
Que vais-je apprendre dans le module 6 sur le monitoring et la sécurité dans le Cloud ?
Vous apprendrez : mise en place de la supervision avec Prometheus et Grafana, suivi de la performance, alertes et troubleshooting, sécurisation des environnements cloud : gestion des utilisateurs, règles IAM, chiffrement, audit de conformité et bonnes pratiques sécurité.

Session 6 - Data Science

Quelle est la durée de la formation 'Data Science & Data Analytics' ?
La formation dure 2 jours et se déroule en ligne de 18h à 21h.
Que vais-je apprendre dans le module 1 sur les essentiels Python et R ?
Vous apprendrez : installation et configuration des environnements RStudio et Jupyter Notebook, structures de données en Python et R, fonctions de base, bibliothèques clés : dplyr, ggplot2 pour R ; pandas, matplotlib pour Python, exercices pratiques de manipulation simple de données.
Que vais-je apprendre dans le module 2 sur la manipulation et le traitement de données ?
Vous apprendrez : nettoyage avancé des données : gestion des valeurs manquantes et doublons, transformation des jeux de données : normalisation, agrégation, utilisation de SQL pour l'extraction et la jointure de données relationnelles, création de pipelines de données simples et automatisés.
Que vais-je apprendre dans le module 3 sur l'exploration et la visualisation des données ?
Vous apprendrez : analyse statistique descriptive avancée, création de dashboards avec Tableau, Power BI, Datawrapper, visualisation interactive : filtres, segmentations, graphiques dynamiques, cas pratique : création d'un dashboard d'indicateurs métiers.
Que vais-je apprendre dans le module 4 sur les statistiques et probabilités appliquées ?
Vous apprendrez : concepts clés : distributions, loi normale, tests d'hypothèse, corrélation, régression linéaire simple et multiple, exercices d'analyse statistique sur datasets métiers, interprétation des résultats et communication aux non-spécialistes.
Que vais-je apprendre dans le module 5 sur le Machine Learning appliqué ?
Vous apprendrez : modèles supervisés et non supervisés adaptés à la data analytique, comment poser et formuler un problème business en termes ML, validation croisée, surapprentissage, métriques clés, atelier pratique : cas concret de prédiction ou segmentation.
Que vais-je apprendre dans le module 6 sur le projet d'analyse complète ?
Vous apprendrez : de l'import des données à la communication des insights, réalisation d'une étude de cas complète : collecte, nettoyage, visualisation et rapport, travail en groupe avec échanges et présentation finale, retour critique et recommandations business.
